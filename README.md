# 🤖 بوت تيليغرام للاشتراكات

## خطوات الإعداد الكاملة

---

### الخطوة 1 — إنشاء البوت

1. افتح تيليغرام، ابحث عن @BotFather
2. اكتب: /newbot
3. اختر اسم للبوت
4. احتفظ بالـ TOKEN الذي سيعطيك إياه

---

### الخطوة 2 — معرفة ID حسابك

1. افتح تيليغرام، ابحث عن @userinfobot
2. اكتب /start
3. سيعطيك رقم ID حسابك (مثل: 123456789)

---

### الخطوة 3 — رفع الكود على GitHub

1. اذهب إلى github.com وأنشئ حساباً مجانياً
2. اضغط New Repository
3. اسم المستودع: telegram-bot
4. ارفع الملفات الثلاثة: bot.py و requirements.txt و railway.toml

---

### الخطوة 4 — تشغيل على Railway

1. اذهب إلى railway.app
2. سجل دخول بحساب GitHub
3. اضغط New Project
4. اختر Deploy from GitHub repo
5. اختر المستودع telegram-bot
6. اضغط على المشروع ثم Variables وأضف:
   - BOT_TOKEN = (التوكن من BotFather)
   - ADMIN_ID = (رقم ID حسابك)
7. اضغط Deploy

---

## أوامر الأدمن (أنت فقط)

| الأمر | الوظيفة |
|-------|---------|
| /admin | لوحة التحكم الرئيسية |
| /broadcast رسالتك | إرسال رسالة لجميع المشتركين |
| /reply ID رسالة | رد خاص على مشترك محدد |
| /setexpiry ID أيام | تجديد عضوية مشترك |
| /deactivate ID | إلغاء عضوية مشترك |

## أوامر المشتركين

| الأمر | الوظيفة |
|-------|---------|
| /start | التسجيل في الخدمة |
| /status | معرفة حالة الاشتراك |
| /contact رسالة | التواصل مع الأدمن |

---

## مثال عملي

**بث رسالة للكل:**
```
/broadcast 🔥 تحليل جديد على الذهب - الهدف 2400$
```

**الرد الخاص على مشترك:**
```
/reply 123456789 مرحباً، اشتراكك تم تجديده
```

**تجديد عضوية لمدة 30 يوم:**
```
/setexpiry 123456789 30
```
